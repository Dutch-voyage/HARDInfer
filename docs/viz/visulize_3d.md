# Guides on Visualizing 3d figures

## Visulize Sparsity Utilzations

Prepare you summary of different sparsity utilizations generated by [./eval/layouts_efficienc](../../eval/layouts_efficiency.py) in a json format file [HA_latency_summary.json](../../HA_latency_summary.json).

Then run with 
```
uv run python ./viz/sparsity_utilization.py 
```

![su](../../figs/sparsity_utilizations.png)

## Visualize Latency Breakdowns regarding Sparsity Utilization

Prepare you latency breakdown data in `sparse_pattern_to_latency.npz` that is generated by [./eval/layouts_efficienc](../../eval/layouts_efficiency.py). 

Then run with 
```
uv run python ./viz/latency_sparsity_utlization.py
```
